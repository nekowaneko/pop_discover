# Pixiv人氣探索工具
## 想解決的問題
1. 自己的作品在pixiv上可以有多高的人氣？需要滿足哪些條件？

2. 依照我想要的標籤，最多可以有多少觀看數？作品製作方向為何？
## 基本想法
1. 輸入圖片 --> 輸出統計的觀看數(含標準差)、十大標籤、主題命名與描述。

2. 輸入標籤 --> 輸出統計的觀看數(含標準差)與前十大人氣的作品。
